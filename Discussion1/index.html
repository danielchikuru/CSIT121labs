Web Design Evaluation

Website Information

  • URL of the website: https://fitmencook.com
  • Name of the website: Fitmencook
  • Target audience: The target audience of the site is people on a healthy diet journey; its purpose is to create budgetfriendly and enjoyable meals.

Site Organization

The site is organized into sections such as Home, Recipes, Shop, Meal Plans, Blog, Cookware, Cookbook, About, and Contact. Each section provides specific information related to its topic.
There is also a more detailled recipe section that has different categories such as chicken, high proteins, salad, low carb, and vegetarian.

CRAP Design Principle

The website uses a simple Contrast principle. Most of the website elements are on a white background with a dark blue font for the texts. However, the website is enhanced by the pictures spreaded thoughout the page layout

Navigation Design

The site mostly uses horizontal menu navigation with sidebar navigation within each sections for easy access to different contents of the website.

Web Design Best Practices Checklist

  1. Clear and Consistent Navigation: The site provides clear navigation options, but the consistency could be improved.
  2. Mobile Responsiveness: The website is responsive and works well on various devices.
  3. Readability: The text is legible and well-structured for easy reading.
  4. Visual Appeal: The website has a modern design with visually appealing elements.
  5. Page Load Speed: The website loads quickly without any noticeable delays.
  6. Call to Action Buttons: The site includes prominent call-to-action buttons for engaging users.

Recommendation

The sections at the top of the website could be improved in a way we see the contents of each sections when hovering them..
